## 2024-05-18 - No Web Frontend
**Learning:** This repository is entirely a ROS 2 backend (C++/Python) for robotics, with no web frontend. Standard UX enhancements (ARIA labels, focus states, UI elements) do not apply, and backend logic must not be altered for "UX" purposes.
**Action:** Always verify the presence of a frontend UI before attempting UX tasks. Abort UX enhancement requests in pure backend/robotics repositories without creating a PR.
